📚 Cara Menggunakan

Tool ini dirancang user-friendly namun powerful untuk menghasilkan prompt berkualitas profesional:

1

Pilih Kategori Nature

Klik salah satu dari 5 tab kategori: Landscape Vista, Botanical Macro, Seasonal Moods, Weather Drama, atau Aerial Perspectives sesuai kebutuhan project Anda.

2

Gunakan Smart Presets (Opsional)

Klik tombol preset untuk langsung apply setting optimal seperti "Mountain Golden Hour" atau "Forest Canopy Texture" - sudah dioptimalkan berdasarkan riset Adobe Stock.

3

Customize Parameters

Sesuaikan dropdown menu untuk subject, lighting, composition, colors, dan technical specs. Atur juga Midjourney V7 settings seperti aspect ratio dan quality.

4

Generate Prompt

Klik "Generate Standard Prompt" untuk AI umum atau "Generate Midjourney V7" untuk Discord Midjourney dengan parameter lengkap.

5

Copy & Use

Copy prompt yang dihasilkan dan paste ke AI tool pilihan Anda. Untuk bulk generation, gunakan fitur "Generate Bulk Prompts" hingga 500 prompts sekaligus.

💡 Pro Tip

Gunakan "Random Current Category" untuk eksplorasi ide kreatif, atau bulk generation untuk project besar yang membutuhkan variasi prompt dalam kategori yang sama.
